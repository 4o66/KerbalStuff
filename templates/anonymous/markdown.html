Kerbal Stuff Markdown

Kerbal Stuff supports vanilla markdown with two exceptions:

  • No inline HTML
  • Added support for embedded things

Example

Here's an example of some markdown and how it would look on Kerbal Stuff:

This is my super cool mod! It's **really** cool.

### Features

* Adds things to the game!
* Removes things from the game!

Which appears like this:

This is my super cool mod! It's really cool.

Features

  • Adds things to the game!
  • Removes things from the game!

There's even more you can do with it. I suggest reading the Markdown spec.

Embedding Content

You can embed YouTube videos or Imgur albums by wrapping their URL in [[brackets]].
